style.css и res/

В файле style.css описываются стили приложения (или отдельной его части). Особенности следующие:

  • Комментарии можно использовать как /*...*/, так и //... - как в JavaScript и C/C++
  • При указании пути к файлу необходимо его предварять #RES#/. Например, background: url(#RES#/header_logo.png)
  • Все файлы, на которые ссылается файл стилей style.css, должны пасполагаться в директории res/, которая располагается на том же уровне, что и сам файл style.css (или иметь там символическую ссылку)
