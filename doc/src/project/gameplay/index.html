gameplay.js (всего проекта)

Внутри gameplay.js должно быть следующее:

... (объявление и реализация классов логики) var $o = {...}; // параметризующий объект /* ниже следует конструкция для условной компиляции: в релиз не пойдёт информация о том, какими инструментами отладки мы пользовались*/ #USAGE_BEGIN#debug## $Traliva.$debug = { $state: true, $url: 'example.com' }; #USAGE_END#debug## $Traliva.$init($o);

Формат передаваемого параметризующего объекта следующий.

Подробнее о классах логики здесь.

Функция $Traliva.$init() не предполагает множественного вызова (вызовы этой функции в подгружаемых частях не считается). При повторном вызове этого метода в консоль должно быть выведено сообщение об ошибке.
