Формат объекта, передаваемого в функцию Traliva.init()

Передаваемый в функцию Traliva.init() объект должен иметь следующие поля:

  • target (String) - это свойство можно не указывать
  • get_layout (функция от (w(int), h(int), target(String)))
  • layouts (Object)
  • widgets (Object)
  • states (Object)
  • initApi (Object)
