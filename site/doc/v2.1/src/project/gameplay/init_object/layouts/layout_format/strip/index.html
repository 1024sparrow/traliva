Strip

Данный лэйаут представляет собой линейный контейнер - в строку или в колонку выстраивает дочерние виджеты (которые, в свою очередь, тоже могут быть контейнерами). Представляется объектом с обязательными свойствами: type, orient и items.

Опционально, данный элемент лэйаута может иметь свойство id - идентификатор, по которому в коде вдальнейшем можно будет получить доступ к данному элементу (Traliva.widgets., имеет тип Traliva.Strip).

Свойство type должно иметь значение 'strip'. По этому свойству фреймворк определяет тип элемента лэйаута.

Свойство orient определяет ориентацию лэйаута: значение 'v' (сокращение от vertical) задаёт расположение элементов в колонку, значение 'h' (сокращение от horizontal) задаёт расположение элементов в строку.

Свойство items - массив с дочерними элементами лэйаута.

Пример:

{ type: 'strip', orient: 'h', items: [ 'my_beautifulWidget', { 'type': 'strip', 'orient': 'v', 'items': [...] }, { 'type': 'stack', ... }, 'const myAnotherBeautifulWidget' ] }
