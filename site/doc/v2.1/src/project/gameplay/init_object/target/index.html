target

идентификатор платформы, под которую будет собираться отладочная версия проекта (бессерверный вариант). По имени директории с собирающими под нужную платформу скриптами. В релизной сборке не используется (используйте условную компиляцию). Например, для сайта здесь может быть 'web', а для мобильного приложения под Android - 'android'.

В функцию get_layout передаётся идентификатор платформы. И если get_layout вернула udefined вместо строкового идентификатора лейаута, то под такую платформу проект собираться не будет.

Это свойство можно не указывать. Ипользуется только в отладочной сборке.
