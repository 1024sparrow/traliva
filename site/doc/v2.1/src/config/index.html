COMPRESS_NAMES=true # убирать пробельные символы в началах строк и символы переноса строки. Настройка не действует, если не активирована CO_RELEASE. COMPRESS_LINEBREAKS=true ## ОПЦИИ КОМПИЛЯЦИИ ************** # наличие отладочной панели(может быть и не задействована, но сам код панели остаётся) и отладочного вывода в консоль CO_DEBUG=true # активирует опции сжатия кода, после применения которых отлаживаться будет затруднительно (удаляются комментарии, переменовываются переменные и функции, убираются переносы строк и отступы). #CO_RELEASE=true # Если включена опция CO_DEBUG, игнорируются опции COMPRESS_LINEBREAKS и COMPRESS_LINEBREAKS

Когда закончите с разработкой JavaScript-овой части, закомментируйте строку с CO_DEBUG и раскомментируйте строку с CO_RELEASE - код резко сократится в размерах.
