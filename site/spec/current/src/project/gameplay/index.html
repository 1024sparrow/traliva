gameplay.js (всего проекта)

Внутри gameplay.js делаем вызов

Traliva.init(o);

Формат передаваемого параметризующего объекта следующий.

Функция Traliva.init() не предполагает множественного вызова (вызовы этой функции в подгружаемых частях не считается). При повторном вызове этого метода в консоль должно быть выведено сообщение об ошибке.
