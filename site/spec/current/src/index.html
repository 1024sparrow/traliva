src

В данной директории располагаются исходники проекта. Перечислим файлы и директории, которые здесь можно найти:

  • pro - файл проекта, который вам понадобится для сборки проекта под заданные платформы с помощью утилиты compiler
  • config - файл параметров сборки. Здесь задаются параметры сборки: включать ли отладочный код в результат сборки, сжимать ли JavaScript (ускоряется загрузка и исполнение, но файл становится нечитаемым - переменные будут переименованы, и весь код будет в одну строку)
  • target_platforms - в этом файле перечисляются названия платформ, под которые нужно осуществлять сборку (по именам соответствующих директорий в src/build_scripts/targets/). Этот файл используется, если в config-файле не указана опция TARGET, которая задаёт единственную платформу, под которую будет осуществляться сборка
  • build_scripts/ - в данной директории находятся скрипты, участвующие в сборке.
  • build_scripts/targets/ - в данной директории располагаются директории со скриптами, генерирующими код под различные платформы. git-подмодуль, уазывающий на github.com/1024sparrow/traliva_platforms или его форк.
  • project - директория с кодом вашего приложения. git-подмодуль, указывающий на github.com/1024sparrow/traliva_example или его форк.
