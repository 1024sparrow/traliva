Отдельные виджеты

Данный тип лэйаута представляется строкой. Указывается уникальный идентификатор виджета, который более подробно описан (конструктор и объект состояния) в свойстве widgets. Идентификатор виджета не может содержать пробельных символов (по ним идёт отсечка специального слова const(см. ниже)).

Перед идентификатором виджета может быть указано специальное слово const. Константность виджетов может проявиться только при использовании механизма подгрузки частей: подгружаемые код не может переопределять константные виджеты - при использовании в подгружаемом лейауте идентификатора, как у константного виджета родительского кода, в случае отсутствия определённого собственного виджета с таким идентификатором в консоль будет выведено сообщение об ошибке, но виджет переопределён не будет.

Примеры:

'my_beautiful_widget', 'const my_const_widget'
