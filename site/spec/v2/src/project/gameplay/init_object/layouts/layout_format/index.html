Формат описания лейаута

Лэйаут представляет собой описание компоновки пользовательских виджетов (которые описываются отдельно в свойстве widgets объекта, передаваемого параметром в функцию Traliva.init()).

Лэйаутом мы называем описание некоего виджета. Этот виджет может быть представлен в виде строкового идентификатора, указывающего на элемент свойства widgets, а может и описывать некую совокупность других лэйаутов.

Лэйауты могут быть следующими:

  • Отдельные виджеты
  • Контейнеры
    • strip
    • stack
    • Пользовательские контейнеры

Пример лэйаута:

Traliva.init({ ... layouts:{ small:{ }, normal:{ // <-- вот он, лэйаут type: 'strip', orient: 'v', items:[ { widget:{ type: 'strip', orient: 'h', items:[ { widget: 'header__bnCatalogue', size: '128px' }, { widget: 'header__bnSaved', size: '128px' }, {}, // stretch { widget: 'header_right__bnAuth', size: '64px' } ] }, size: '64px' }, { widget:'main_scene' }, { // footer widget: 'footer_stub', size: '64px' } ] }, large:{ } }, ... });
