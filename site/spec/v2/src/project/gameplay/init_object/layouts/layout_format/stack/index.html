Stack

Данный лэйаут представляет собой многослойный контейнер - дочерние элементы лэйаута располагаются один поверх другого. Представляется объектом с обязательными свойствами type и items.

Опционально, данный элемент лэйаута может иметь свойство id - идентификатор, по которому в коде вдальнейшем можно будет получить доступ к данному элементу (Traliva.widgets., имеет тип Traliva.Stack).

Свойство type должно иметь значение 'stack'. По этому свойству фреймворк определяет тип элемента лэйаута.

Свойство items - массив с дочерними элементами лэйаута.

Пример:

{ type: 'stack', items: [ 'my_beautifulWidget', { 'type': 'strip', 'orient': 'v', 'items': [...] }, { 'type': 'stack', ... }, 'const myAnotherBeautifulWidget' ] }
