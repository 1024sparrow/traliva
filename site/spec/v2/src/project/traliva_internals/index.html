Внутри traliva.js

Исходный код фреймворка сгруппирован по четырём группам:

  • states - базовые классы для работы с объектом состояния
  • widgets - базовые классы для построения виджетов
  • utils - отдельные функции
  • init - здесь реализована функция Traliva.init(), а также здесь находятся базовые классы WidgetStateSubscriber и LogicsStateSubscriber, которые не имеют смысла за пределами функции Traliva.init().
